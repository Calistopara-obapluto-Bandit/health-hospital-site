# Health - Medical Hospital Website

A responsive medical hospital website template built with Bootstrap, HTML5, and CSS3.

## Features

- Fully responsive design (desktop, tablet, mobile)
- Multi-language support (English, Greek, Ukrainian)
- Hospital departments showcase
- News & updates section
- Online appointment booking
- 24/7 emergency services information
- Contact information and location map
- Modern UI with animations

## Files Structure

```
├── index.html              # Main page
├── news-detail.html        # News detail page
├── css/                    # Stylesheets
├── js/                     # JavaScript files
├── images/                 # Image assets
├── fonts/                  # Font files
└── README.md              # This file
```

## Deployment on Render

### Step 1: Prepare for Upload
- All files are ready in this folder
- No build process needed

### Step 2: Deploy on Render
1. Visit [render.com](https://render.com)
2. Create account or sign in
3. Click **New +** → **Static Site**
4. Select **Public Git Repository** or upload manually
5. For manual upload:
   - Download this folder as ZIP
   - Drag & drop into Render
6. Set Publish directory: `/`
7. Click Deploy

Your site will be live in minutes!
